Данный набор программного обеспечения (далее называемый
"Tkabber-Pack") состоит из нескольких программных пакетов,
являющихся свободным программным обеспечением и
предназначенных для совместной работы друг с другом. Это:
* Tkabber -- клиент протокола XMPP (Jabber). Распространяется
  по лицензии GNU General Public License.
* Tcl и Tk -- Tool Command Language и его библиотека
  поддержки графического интерфейса пользователя Tk.
  Распространяется по собственной лицензии в стиле BSD.
* Несколько сторонних расширений к Tcl/Tk: (части) Tcllib,
  BWidget, tls, tcludp, tclmore, ztcl, tclWinidle, Tkinactive,
  Winico, Snack, TkImg, Memchan, Trf, vfs.
* Некоторые из этих расширений используют код свободных библиотек
  (будучи скомпонованы с ними статически или
  динамически): OpenSSL и zlib.

Полные тексты лицензий свободного ПО, составляющего данный
набор, будут доступны после инсталляции в каталоге
"doc\licenses", находящимся в директории, в которую будет
установлен этот пакет.

Сам Tkabber-Pack распространяется по лицензии GNU General
Public License. Её полный текст находится в файле "GNU_GPL.txt",
который будет доступен в уже упоминавшемся каталоге
"doc\licenses" после инсталляции; также он доступен в сети
Интернет по адресу http://www.gnu.org/licenses/licenses.html#GPL
Данная лицензия покрывает все прочие файлы, не входящие в пакеты
программного обеспечения, описанные выше, и относящиеся
непосредственно к инсталлятору, а именно: скрипт для генерации
инсталлятора Tkabber-Pack, соответствующие графические файлы
и прочие файлы поддержки.
Исходный код Tkabber-Pack доступен в сети Интернет по адресу
http://tkabber.jabber.ru/files/tkabber-pack/tkabber-pack-0.11.1.tar.bz2

Используя и/или распространяя Tkabber-Pack, вы соглашаетесь со
всеми лицензиями, по которым распространяются компоненты, его
составляющие, а также он сам.
